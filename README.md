# ⭐ MiniProject FoodTruck-Manager ⭐
## 푸드 트럭 운영 & 예약 관리
> 푸드 트럭 운영 & 예약 관리 웹 페이지

## 📢 개요 
🔹 이동형 푸드 트럭의 **영업 스케줄 공개**, 고객의 **사전 예약(픽업 시간 지정)** 및 **현장 주문**을 지원하는 경량 웹 서비스 구축, 운영자는 스케줄·메뉴·품절·예약을 한 곳에서 관리하고, 고객은 줄 서지 않고 픽업
1. 줄서기 최소화
2. 운영 효율(품절/재고 플래그, 판매 현황)
3. 운영지 공개(주소/좌표 기반 탐색)
4. 모의 결제 흐름(취소/환불 상태 전이)

👉 **대상 사용자**
- **고객(User)**: 근처 트럭 스케줄/메뉴 확인 → 시간 지정 예약 → 현장 픽업/결제(모의)
- **운영자(Owner)**: 본인 트럭/스케줄/메뉴/품절/예약 관리, 현장 주문 기록
- **관리자(Admin)**: 다수 트럭/운영자 계정/권한 관리, 강제 취소/환불, 모니터링
